Chat Widget Embed Demo

This page demonstrates how to embed the chat widget in different ways.

Method 1: Auto-mount (Default)

The widget will automatically appear when the script loads.

<script src="dist/my-widget.umd.js"></script>

Method 2: Manual Mount

Mount the widget with custom configuration.

// Mount with default settings ChatWidget.mount(); // Mount with custom configuration ChatWidget.mount({ position: 'bottom-left', theme: 'dark', title: 'Custom Title', buttonText: 'Get Help' });

Method 3: In a Specific Container

Mount the widget in a specific HTML element.

Widget will appear here

// Mount in specific container ChatWidget.mount({ containerId: 'custom-container', position: 'bottom-right', title: 'Container Widget' });
