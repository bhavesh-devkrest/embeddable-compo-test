Quick Start

Add this script tag to your HTML:

<script src="https://your-vercel-domain.vercel.app/my-widget.umd.cjs"></script>

The widget will automatically appear in the bottom-right corner.

Custom Configuration

// Mount with custom settings ChatWidget.mount({ position: 'bottom-left', theme: 'dark', title: 'Need help?', buttonText: 'Start Chat', onChatStart: () => { console.log('Chat started!'); } });
