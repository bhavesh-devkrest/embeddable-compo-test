Welcome to Our Example Site

This page demonstrates how to integrate the deployed chat widget into a real website. The widget should appear in the bottom-right corner automatically.

Integration Code

Here's the code used to embed the widget on this page:

<!-- Basic embed (auto-mounts) --> <script src="https://your-vercel-domain.vercel.app/my-widget.umd.cjs"></script> <!-- Custom configuration --> <script> ChatWidget.mount({ position: 'bottom-right', theme: 'light', title: 'Need help?', buttonText: 'Start Chat', onChatStart: () => { console.log('Chat started!'); // Add your custom logic here } }); </script>
